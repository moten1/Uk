# AuraDx Wellness App

## Setup

1. Install dependencies
```
npm install
```

2. Run the development server
```
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy

Push to GitHub and deploy on Vercel for instant cloud hosting.
